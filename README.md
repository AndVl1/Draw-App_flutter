# Flutter Drawing app

Flutter drawing app for applying to JB internship.

Tested on Android, iOS emulators and in browser (in addition to the main task).

-----------------------
## Features

- Draw with finger
- Draw lines
- Draw rectangles
- Erase drawings
- Choose color
- Clear canvas
- Select stroke width
